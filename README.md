# 🌌 NIJJU CYBER-EDITOR

A premium, real-time collaborative code editor with a **Midnight Cyber-Noir** aesthetic. Built for developers who demand both high-performance functionality and cutting-edge design.

![Platform Preview](https://img.shields.io/badge/Aesthetic-Cyber--Noir-00d2ff?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-React%20%7C%20Node.js%20%7C%20Socket.io-020617?style=for-the-badge)

## ✨ Core Features

### 🛠️ Collaborative Workspace
- **Real-time Sync**: Multi-user code synchronization powered by Socket.io.
- **Dynamic File Naming**: Edit filenames on the fly directly from the editor tabs.
- **Peer List**: See active collaborators with real-time status indicators.

### 🧠 Intelligent Utils
- **AI Engine**: Integrated structural analysis (Nijju-AI Link).
- **Multi-Language Support**: Seamlessly switch between Python, JavaScript, Java, C++, and more.
- **Console Suite**: Fully partitioned STDIN/STDOUT for professional debugging.

### 💎 Premium Aesthetics
- **Cyber-Noir Theme**: High-contrast midnight black workspace with electric blue neon accents.
- **Glassmorphism**: Elegant translucent overlays and frosted-glass modal interfaces.
- **Animated Mascot**: 3D Cyber-Robot mascot with floating micro-animations.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd FRONTEND-LEETCODE
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Matrix**
   ```bash
   npm run dev
   ```

### Environment Variables
Create a `.env` file in the root directory:
```env
REACT_APP_SERVER_URL=http://localhost:5000
```

## 📂 Project Structure

- `/src/pages`: Core application views (Home, Editor, Language Selection).
- `/src/components`: Reusable UI modules (Code Editor, Modals, Peer Cards).
- `/src/languages.js`: The central language metadata database.
- `/public/assets`: High-quality 3D assets and mascots.

## 🛡️ Security & Performance
- **Docker-Ready**: Designed for sandboxed code execution.
- **Optimized Styles**: Custom Vanilla CSS for maximum performance and zero bloat.
